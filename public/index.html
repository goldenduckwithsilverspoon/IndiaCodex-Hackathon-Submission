üì° API Endpoints

POST /api/upload

Upload files to the CDN. Returns CID for content addressing.

curl -X POST -F "file=@image.jpg" /api/upload

GET /content/:cid

Retrieve content by CID. Immutable, cached, globally distributed.

curl /content/bafkreih4rke...

POST /api/storage-proof/:cid

Generate Merkle proof for blockchain verification.

curl -X POST /api/storage-proof/:cid

GET /api/stats

Get node statistics: requests, bandwidth, uptime, peers.

curl /api/stats
